# 📱 NEÜ Yoklama Sistemi — Mobil Uygulama

QR kod ve GPS konum doğrulama tabanlı üniversite yoklama sistemi mobil uygulaması.

## 🚀 Özellikler
- 📷 Dinamik QR kod okuma (5 dakikada geçersiz olur)
- 📍 GPS konum doğrulama ile yoklama suistimalini önleme
- 🔔 Gerçek zamanlı yoklama bildirimleri
- 📊 Öğrenci yoklama geçmişi görüntüleme
- 🔐 Firebase kimlik doğrulama

## 🛠️ Kullanılan Teknolojiler
- **Flutter** — Cross-platform mobil uygulama
- **Firebase** — Veritabanı ve kimlik doğrulama
- **Dart** — Programlama dili

## ⚙️ Kurulum
```bash
git clone https://github.com/Umut8001/neu_yoklama_sistemi.git
cd neu_yoklama_sistemi
flutter pub get
flutter run
```
